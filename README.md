# Olist-Ecommerce-SQL-Analysis
End-to-end SQL analysis of an e-commerce dataset using PostgreSQL, covering customer behavior, revenue trends, logistics performance, and advanced analytics with CTEs and window functions.

## Tools Used
- PostgreSQL
- pgAdmin
- SQL (CTEs, Window Functions, Aggregations)
- Markdown for documentation

## Analysis Breakdown
### Level 1: Basic Analysis
- Customer base and order overview
- Order status distribution
- Top cities, categories, and payment methods
- Revenue and Average Order Value (AOV)

### Level 2: Intermediate Analysis
- Monthly revenue and seasonality
- State-wise and category-wise revenue
- Repeat customer identification
- Delivery time and late delivery analysis
- Freight cost and cancellation analysis

### Level 3: Advanced Analysis
- Top customers by spend
- Month-over-month revenue growth
- Repeat customer revenue contribution
- Category ranking by month
- Running total revenue
- Top products per category
- Customers with above-average order value

## Key Business Insights
- Repeat customers contribute a disproportionate share of revenue.
- Revenue shows strong seasonal patterns.
- A small number of orders significantly increase freight costs.
- Certain product categories demonstrate consistent demand across months.

## Outcome
This project demonstrates end-to-end SQL analysis capability, from data exploration to advanced business insights, suitable for Data Analyst and Business Analyst roles.
