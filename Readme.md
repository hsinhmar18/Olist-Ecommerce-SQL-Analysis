# Dataset Information

This project uses the **Olist Brazilian E-commerce Dataset**, a publicly available dataset containing real-world e-commerce transactions from Brazil.

## Dataset Overview
The dataset includes information related to:
- Customers
- Orders
- Order items
- Products
- Payments
- Logistics and delivery timelines

Only the tables required for analysis have been used in this project.

## Dataset Source
- Platform: Kaggle  
- Dataset Name: Olist Brazilian E-commerce Dataset

## Usage Note
Raw CSV files are not included in this repository to keep it lightweight and focused on SQL analysis.

Users can download the dataset directly from Kaggle and recreate the database schema using the provided `schema.sql` file.

## Purpose
The dataset is used strictly for **educational and analytical purposes**, including:
- SQL querying practice
- Business insight generation
- Portfolio demonstration for Data Analyst and Business Analyst roles
